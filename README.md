# Converent.ca Website

Standalone Next.js website for Converent.

## Local development

```bash
npm install
npm run dev
```

## Edit perspectives with CMS (free)

This project includes a local Keystatic CMS (no subscription required).

- Open `http://localhost:3000/keystatic` while running `npm run dev`
- Edit entries in the `Perspectives` collection
- Content is stored as files in `content/perspectives/`
- Commit and push content changes like normal code updates

## Production build

```bash
npm run build
```
